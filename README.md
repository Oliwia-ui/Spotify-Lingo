# Spotify-Lingo

Spotify Lingo is a language-learning mini game:
- Search songs using the Jamendo API
- Filter by lyrics language
- Play a song
- Fill in missing words from real lyrics (mini game)

This repo is split into:
- `/client` = React (Vite) frontend
- `/server` = Node/Express backend (API + Jamendo integration)

---

## ✅ Requirements
- Node.js installed (recommended: v18+)
- A Jamendo Developer Client ID

---

## 🔑 Jamendo Client ID (Required)
Each team member needs a local `.env` file for the backend (we do NOT push API keys to GitHub).

Create your Jamendo Client ID here:
https://developer.jamendo.com
